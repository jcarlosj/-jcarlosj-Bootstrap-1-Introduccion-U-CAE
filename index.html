Button tags

La clase btn esta diseñada para se usada con el elemento button Sin embargo también se puede aplicar a los elementos a, input

Aunque algunos navegadores pueden aplicar representaciones ligeramente diferentes a las que deseamos, podemos ver como la clase btn en un elemento a puede ser usado para activar una funcionalidad de la página (como colapsar el contenido), en lugar de enlazar a otras o secciones dentro de la actual, a este tipo de enlaces se debe asignar un rol rol="button" para transmitir el proósito a las tecnologías de asistencia como lectores de pantalla

Button style

Bootstrap incluye varios estilos de botones predefinidos, cada uno con su propio propósito semántico, con algunos extras incluidos para un mayor control.

Las clases que permiten esto son: btn-primary, btn-secondary, btn-success, btn-danger, btn-warning, btn-info, btn-light, btn-dark

Outline buttons

La clase .btn-outline-* elimina todas las imágenes de fondo y los colores de cualquier botón.

Tamaños

Las clases btn-sm, btn-lg esta predeterminadas para cambiar el tamaño de los botones

También usando la clase btn-block podemos hacer que el botón se comporte como un elemento de bloque
