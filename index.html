Button Plugin

Se puede controlar los estados de los botones o crear grupos de botones para más componentes como barras de herramientas.

Toggle states Estado de alternacia

Se debe agregar data-toggle="button" para alternar el estado de un botón activo. Si está activando un botón previamente, debe agregar manualmente la clase .active y aria-pressed="true" al elemento button
